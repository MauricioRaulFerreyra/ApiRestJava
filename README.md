
## Proyecto estructura básica APIRestFull con Spring Boot

Este proyecto se basa en la creación de una API Rest (interfaz de programación de aplicaciones que se ajusta a la arquitectura Rest) con Java, utilizando Spring-Boot , que es una herramienta para el desarrollo de aplicaciones web y microservicios con Spring Framework.
Es una aplicación CRUD sencilla, en la cual se utiliza para la base de datos MySQL Workbench, Maven (herramienta de gestión de proyectos que se utiliza para el manejo de dependencias, como herramienta de compilación e incluso como herramienta de documentación).

#### Se crea el proyecto con el link: https://start.spring.io/
